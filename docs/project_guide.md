# Project Guide: sleepylee02.github.io

This document serves as a central knowledge base for understanding and managing your Hugo-based portfolio website.

## 1. Project Overview
- **Type**: Static Site Generator (Hugo)
- **Theme**: `terminal` (A simple, retro-style theme)
- **Deployment**: Likely GitHub Pages (based on the repository name)

## 2. Directory Structure & Key Files

| Directory/File | Description |
| :--- | :--- |
| **`hugo.toml`** | **Main Configuration File**. Controls site title, menu links, theme settings, and parameters. |
| **`content/`** | **Page Content**. Contains the Markdown files (`.md`) that become your website pages. |
| **`themes/terminal/`** | **Theme Files**. The raw code for the design. **Avoid editing inside here directly.** |
| **`assets/`** | **Global Assets**. Stores CSS and JS that processing pipes can use. |
| **`static/`** | **Static Files**. Images, favicons, etc., served exactly as-is. |
| **`layouts/`** | **Layout Overrides**. (Currently empty) Place HTML files here to override theme templates. |
| **`public/`** | **Build Output**. Generated by the `hugo` command. This is what gets deployed. |

## 3. Functionality & Scripts

### JavaScript (`themes/terminal/assets/js/`)
The site uses minimal JavaScript, primarily for two features:

1.  **`code.js`**: 
    -   Finds all code blocks on the page.
    -   Adds a "Copy" button to them.
    -   Displays the language usage (e.g., "Python", "JavaScript").
    
2.  **`menu.js`**:
    -   Manages the responsive navigation menu.
    -   Handles opening/closing dropdowns.
    -   Closes the menu if you click outside or resize the window.

### Configuration (`hugo.toml`)
-   **Menu**: defined under `[[menu.main]]`. Entries link to `/about`, `/projects`, etc.
-   **Params**:
    -   `themeColor = "orange"`: Sets the primary accent color.
    -   `contentTypeName = "posts"`: default generic content type.

## 4. How to Manage Content

### Adding a New Page
Create a file in `content/` with a standard frontmatter header:
```markdown
---
title: "My New Page"
date: 2023-10-27
draft: false
---

Write your content here in Markdown...
```

### Editing Menus
Open `hugo.toml` and find the `[[menu.main]]` sections. You can change names, URLs, or add new items by copying an existing block.

## 5. Development Workflow

1.  **Start Local Server**:
    ```bash
    hugo server
    ```
    Access at: `http://localhost:1313/` (Live reloads on save).

2.  **Build for Production**:
    ```bash
    hugo
    ```
    Creates the `public/` folder.
